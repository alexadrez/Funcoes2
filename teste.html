Governança de Dados - Metadados CONCLUÍDA

Metadados - Regras de Negócio

  • DESCRIÇÃO (pt-BR / en-US / es-ES): Item em Risco de Ruptura / Item at Risk of Out of Stock / Item en Riesgo de Falta 

  • CONCEITO: O Item em Risco de Ruptura representa o produto o qual ainda tem estoque maior do que zero, porém devido à analise de sua demanda e estoque apresenta indícios de que irá entrar em ruptura nos próximos dias.

  • PROPÓSITO: O propósito é identificar produtos que estão em risco de ruptura para que haja uma atuação de reposição.

  • OBSERVAÇÕES: Não se aplica.

Metadados - Forma de Uso

  • FORMA DE USO: O campo é uma agregação, portanto deve ser calculado em tempo de execução nos produtos.
  • OBSERVAÇÕES: Quando for um produto direcionado ao varejo, mapear os campos necessários para o cálculo no RDW. Quando for direcionado à indústria, mapear os campos necessários no CDM.

Metadados - Fórmulas e Cálculos

  • FÓRMULA EM TEMPO DE EXECUÇÃO (bl_item_tvd): SE bl_out_of_stock = '0' E vl_tvd > '0' ENTÃO '1', SENÃO, '0'.
  • OBSERVAÇÕES: No quadro Exemplo de Consultas de Dados para Validação há uma query com um exemplo de consulta para validação.

Exemplo de Consultas de Dados para Validação

Icon
OBSERVAÇÕES: Estes são apenas exemplos de consultas. Os desenvolvedores tem a flexibilidade de implementar as agregações de acordo com seus conhecimentos nas melhores práticas de performance em cada tecnologia utilizada.
  • SELECT F.id_time, F.id_item, nm_item, F.id_location, nm_location, bl_out_of_stock, vl_tvd,
  • CASE WHEN bl_out_of_stock = 0 AND vl_tvd > 0 THEN 1 END AS 'bl_item_tvd'
  • FROM RDW_CARREFOUR.FACT_POINT_OF_SALE F WITH(NOLOCK)
  • JOIN RDW_CARREFOUR.DIM_ITEM I WITH(NOLOCK) ON F.id_item = I.id_item
  • JOIN RDW_CARREFOUR.DIM_LOCATION L WITH(NOLOCK) ON F.id_location = L.id_location
  • WHERE id_time = '2021-02-01'
  • AND F.id_location = 114
  • ORDER BY 7 DESC

QUANDO CONSIDERAMOS O TVD, SEMPRE DEVEMOS UTILIZAR A ÚLTIMA POSIÇÃO PARA O PERÍODO DESEJADO

  • SELECT id_time, COUNT(CASE WHEN bl_out_of_stock = 0 AND vl_tvd > 0 THEN vl_tvd END) AS 'Qtd Itens em Risco de Ruptura'
  • FROM RDW_CARREFOUR.FACT_POINT_OF_SALE WITH(NOLOCK)
  • WHERE id_time BETWEEN '2021-02-01' AND '2021-02-06'
  • AND id_location = 114
  • GROUP BY id_time
  • ORDER BY 1 DESC

  • SELECT [Qtd Itens em Risco de Ruptura] AS 'Qtd Itens em Risco de Ruptura'
  • FROM (SELECT id_time, COUNT(CASE WHEN bl_out_of_stock = 0 AND vl_tvd > 0 THEN vl_tvd END) AS 'Qtd Itens em Risco de Ruptura',
  • ROW_NUMBER() OVER (ORDER BY id_time DESC) 'ROW_ID'
  • FROM RDW_CARREFOUR.FACT_POINT_OF_SALE WITH(NOLOCK)
  • WHERE id_time BETWEEN '2021-02-01' AND '2021-02-06' AND id_location = 114
  • GROUP BY id_time) P WHERE ROW_ID = 1
